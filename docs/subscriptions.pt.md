# Subscrição de imagens

O Quor é um catálogo de imagens de containers, e seu modelo de uso é baseado em subscrição de imagens.

Ao subscrever uma imagem do catálogo, você libera o acesso para fazer pull de qualquer versão e qualquer tag associada a essa imagem.

Para realizar a subscrição é necessário ter um plano ativo, atualmente disponível como Trial ou Enterprise.

## Gestão de imagens subscritas

No painel do Quor, em `Settings` > `Organization`, você pode visualizar todas as imagens às quais sua organização subscreveu.

Nessa página também é possível acompanhar:

- Data de subscrição e usuário;
- Plano ativo e vigência.
  
Para acessar esses detalhes, você precisa ter uma conta válida e plano ativo: [Quor](https://app.quor.dev/settings)
